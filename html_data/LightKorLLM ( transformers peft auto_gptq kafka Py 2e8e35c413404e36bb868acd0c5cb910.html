프로젝트 보고서 [2023년 10월]

LightKorLLM 프로젝트는 일반 상용 그래픽 카드 (RTX 30시리즈, GTX 16시리즈 등)에서 사용할 수 있는 LLM 프로젝트를 목표로 만들어졌으며,

소규모 개인 프로젝트 또는 학부 프로젝트 내부에서 활용할 수 있도록 파이선 Flask 서버 형태의 백엔드 서버와 kafka 를 활용한 메세지 기반 프로토콜 또한 지원 합니다.

또한 차후 해당 프로젝트를 서브 시스템으로 하는 LLM 파이프라인 시스템 구축을 준비하면서 해당 프로젝트를 위한 Docker Image 배포 및 Dockerfile을 통한 자동화 시스템을 도입하면서 Docker 시스템에 대해 좀더 세부적으로 학습하고 활용하게 되는 계기를 가지게 되었으며,

또한 이 프로젝트를 진행하면서 가장 어려웠던 점은 양자화된 상태의 모델이 생성하는 답변을 적절한 길이로 조절하는 문제이였습니다.

단편적인 예시로

###사용자 : 미국 독립전쟁에 대해 알려줘.

###AI :

를 입력값으로 LLM의 추론 결과값을 생성한 결과가.

###사용자 : 미국 독립전쟁에 대해 알려줘.

###AI : 미국 독립전쟁은 ~~~

###사용자 : 전쟁의 역사에서~~

위와 같이 AI의 답변내용을 넘어 사용자의 질문조차 생성하는 케이스가 수도없이 발생하게 되어 해당 문제를 해결하기 위해 prompt 튜닝 및 추가 데이터를 활용한 PEFT 학습 등 다양한 방법을 시도해 보았으나, 결국 transformers의 라이브러리에서 제공하는 Stopwords 기능을 통해 문제를 해결하였던 기억이 남는 프로젝트 입니다.
